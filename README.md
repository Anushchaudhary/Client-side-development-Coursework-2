## 📖 Project Description
This project is a fully responsive multi-page real estate website built using HTML5, CSS3, JavaScript, and jQuery. The website simulates a property agency called **Prime Estates**, allowing users to browse property listings, filter results, and submit enquiries through a validated contact form.

The project was developed as part of the COM109 module to demonstrate front-end development skills including UI design, interactivity, and client-side scripting.

---

## 🌐 Pages Included
- Home Page (index.html)
- Properties Page
- About Page
- Contact Page

---

## ⚙️ Features

### 🎯 Home Page
- Hero section with call-to-action
- Animated statistics counter
- Featured property preview cards

### 🏡 Properties Page
- Property listing display
- Filter system (price, type, bedrooms)
- Interactive property cards (expand/collapse)

### ℹ️ About Page
- Company information
- Team section
- Core values layout

### 📩 Contact Page
- Fully validated contact form
- Real-time error handling
- Success message after submission
- Local storage greeting feature

## 🎨 Design Approach

## 🔧 How to Run the Project
1. Download or clone the repository
2. Open the `index.html` file in any web browser
3. Navigate through the website using the menu
